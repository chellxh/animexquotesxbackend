# ANIMExQUOTESxBACKEND

## Summary

This project is a backend API of anime shows, characters, and quotes. The purpose of this project is to help find any quote from any anime.

### Installing

Run in terminal
`npm i`

You can make get requests to this API using:

- /characters
- /characters/id
- /shows
- /shows/id
